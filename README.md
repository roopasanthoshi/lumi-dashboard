# Lumi AI Dashboard

Lumi AI Dashboard is a modern AI-powered productivity dashboard built using React and Vite.  
The application combines multiple productivity and utility features into a single clean and responsive interface.

---

## Features

- AI Daily Brief using Gemini API
- Live Weather Updates
- News Dashboard
- Task Management Board
- Calendar Board
- GitHub Profile Integration
- Responsive Modern UI
- Dynamic Greeting System
- Real-time API Integrations

---

## Tech Stack

- React.js
- Vite
- JavaScript
- Tailwind CSS
- Gemini API
- Weather API
- News API
- GitHub API
